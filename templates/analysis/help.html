{% translate "2. 単一バリアント解析" %}

{% translate "1 つのバリアントを ACMG 2015 + ClinGen SVI 基準で分類します。" %}

{% translate "入力形式" %}

{% translate "次のいずれの形式でも入力できます（MANE Select transcript に限定して解釈します）。" %}

  • {% translate "genome 座標" %}: chr17:7674221G>A
  • {% translate "cDNA" %}: TP53:c.742C>T {% translate "または空白区切りで" %} TP53 742C>T
  • {% translate "protein" %}: TP53:p.R248W {% translate "または空白区切りで" %} TP53 R248W

{% translate "アセンブリ (GRCh38 / GRCh37) を選び、必要なら入力種別を指定します（通常は自動判定で問題ありません）。" %}

{% translate "候補の選択" %}

{% translate "変換結果が複数候補ある場合は一覧から該当するバリアントを選択します。MANE Select で座標が得られない場合は、全 transcript 候補にフォールバックして提示します。" %}

{% translate "結果の見方" %}

  • {% translate "ACMG 2015 と Bayesian の 2 方式の分類、および Bayesian スコアを表示します。" %}
  • {% translate "各クライテリアの判定 (MET / not met / suppressed)・方向・strength・points・根拠を一覧で確認できます。" %}

{% translate "手動編集と再分類" %}

  • {% translate "クライテリアごとに「編集 strength」を選び、根拠を入力して「編集を適用して再分類」を押すと、その場で再計算します。" %}
  • {% translate "「（変更なし）」のままの項目は自動判定を保持します。" %}
  • {% translate "手動編集による再分類の結果はデータベースに保存されません" %}{% translate "（元の自動解析結果は保存されます）。" %}

{% translate "結果のダウンロード" %}

  • {% translate "「JSON でダウンロード」「TSV でダウンロード」から保存できます。" %}
  • {% translate "手動編集を行った場合は、画面に表示されている（編集後の）内容がそのままダウンロードされます。" %}

{% translate "3. バッチ解析 (VCF)" %}

  • {% translate "VCF ファイル (.vcf / .vcf.gz、上限 50MB) をアップロードし、アセンブリを選んで投入します。" %}
  • {% translate "解析は投入順に直列処理されます。ジョブ状態画面で進捗を確認でき、完了すると結果 TSV をダウンロードできます。" %}
  • {% translate "過去のジョブは「ジョブ履歴」から確認できます。" %}
  • {% translate "アップロードファイルと結果ファイルは保持期間 (既定 1 時間) を過ぎると自動削除されます。必要な結果は早めにダウンロードしてください。" %}
